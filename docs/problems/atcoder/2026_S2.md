## AtCoder Beginner Contest [#446](https://atcoder.jp/contests/abc446)

E - Multiple-Free Sequences

由題目可以知道:

\[
\begin{array}{ll}
    S_1 = y, S_2 = x \rightarrow (S_2, S_1) = (x, y) \\
    (S_2, S_1) \rightarrow (S_3, S_2) = (Ax + By, x)
\end{array}
\]

由上面可以得到矩陣轉移方程:

$$
\begin{bmatrix}
A && B\\
1 && 0
\end{bmatrix}
\begin{bmatrix}
S_2 \\
S_1
\end{bmatrix}
=\begin{bmatrix}
AS_1 + BS_2\\
S_1
\end{bmatrix}
=\begin{bmatrix}
Ax + By\\
x
\end{bmatrix}
=\begin{bmatrix}
S_3\\
S_2
\end{bmatrix}
$$

$$
\begin{bmatrix}
A && B\\
1 && 0
\end{bmatrix}
\begin{bmatrix}
S_{i+1} \\
S_i
\end{bmatrix}
=\begin{bmatrix}
AS_{i+1} + BS_{i}\\
S_{i+1}
\end{bmatrix}
=\begin{bmatrix}
S_{i+2}\\
S_{i+1}
\end{bmatrix}
$$

簡單來說每次的轉移會是:

$$(x, y) \rightarrow (Ax + By, x)$$

題目所求 $S_i$ 不能為 M 的倍數因此，可以透過 $\pmod{M}$ 將問題轉化為，求 $S_i \pmod{M} \neq 0$ 的點對個數。

$$(x, y) \rightarrow (Ax + By \pmod{M}, x)$$

由題目的限制可以知道 $0\le x,y < M \le 10^3$。最多會有 $m^2$ 種點對，且當 $(Ax + By \pmod{M}, x)$ 時可以知道一定會是 $m^2$ 種的其中一種。

對於每次 $(x, y) \rightarrow (Ax + By \pmod{M}, x)$ 可以建一個反邊。最後由 $(u, 0), (0, v), (0, 0)$ 做 dfs 把所有遍歷到的節點就是題目所求。


F - Reachable Set 2

VP的時候寫了一個DFS結果TLE，然後改BFS就過了，atcoder 寫dfs的時候要注意!

盡量用 global(少用lambda?)，以及使用陣列，和 vis 去重，不要使用set。

```cpp
int nbr[MX], vis[MX];
vector<int> g[MX];
int cnt_nbr = 0, cnt_vis = 0;
void dfs(int x, int limit) {
    vis[x] = 1;
    cnt_vis++;
    for(auto &y: g[x]) {
        if(y < limit && !vis[y]) {
            dfs(y, limit);
        }
        if(!nbr[y]) {
            nbr[y] = true;
            cnt_nbr++;
        }
    }
};

void solve() {
    int n, m, u, v;
    cin >> n >> m;
    for(int i = 0; i < m; ++i) {
        cin >> u >> v;
        if(u == v) continue;
        g[u].push_back(v);
    }    

    for(int i = 1; i <= n; ++i) {
        if(i == 1) {
            nbr[i] = 1;
            cnt_nbr++;
        }
        if(nbr[i]) dfs(i, i);
        if(cnt_vis == i) cout<<cnt_nbr - cnt_vis<<endl;
        else cout<<-1<<endl;

    }
}
```

## AtCoder Beginner Contest [#452](https://atcoder.jp/contests/abc452)

D - No-Subsequence Substring

這題要計算所有在 $S$ 中不包含構成 $T$ 子序列 t 的子字串個數。

可以透過計算全部方案減去包含 $T$ 子序列的子字串 $s_T$ 的數量: $all - cnt(s_T)$

當固定右端點時，可以觀察到越往左越有機會能夠找到包含 T 的子字串 $s_T$，而一旦找到 $s_T$ 則 $s_T$ 左邊的端點也可以與當前右端點構成 $s_T$。

可以向右遍歷陣列，不斷計算更新可以構成 $s_T$ 的最近位置。

```cpp
void solve() {
    string s, t;
    ll ans = 0;
    cin >> s >> t;
    int n = t.size();
    ll m = s.size();
    vector<int> cnt(n);
    for(int j = 0; j < m; j++) {
        auto &c = s[j];
        int ch = c - 'a';
        for(int i = n - 1; i >= 0; --i) {
            if(t[i] == c) {
                if(i == 0) {
                    cnt[i] = j + 1;
                    continue;
                }
                cnt[i] = cnt[i - 1];
            }
        }
        ans += cnt[n - 1];
    }
    cout<<m * (m + 1) / 2 - ans<<endl;
}
```

F - Exactly K Steps 2

由 Floyd 算法出發，可以知道:
```
for k = 1 to n
    for i = 1 to n
        for j = 1 to n
            M'[i][j] = min(M'[i][j], M[i][k] + M[k][j])
```

由 i 經由 k 至 j，產生出來的矩陣為 $D'$，$D[i][k]$ 為從 i 至 k 為走一步的花費，則 $D'[i][j]$ 為從 i 至 j 走兩步的花費。

由矩陣乘法可以得到

```
for k = 1 to n
    for i = 1 to n
        for j = 1 to n
            D'(i, j) += D(i, k) * D(k, j)
```

可以發現 Floyd 跟矩陣乘法很相似，但是要使用矩陣乘法要滿足 "結合律" 和 "交換律"，假設 $D'(i, j) = min(D(i, k) + D(k, j))$ 的 $min$ 操作表示為 $\oplus$

1. 結合律
   $(D_{a} \oplus D) \oplus D_{b} = D_{a} \oplus (D \oplus D_{b})$

2. 交換律
    $D_{a} \oplus D_{b} = D_{b} \oplus D_{a}$


主要觀察 $min$ 操作是否能夠符合結合律和交換律，若是可以就能使用矩陣快速冪計算。

```cpp
const ll infll = LLONG_MAX / 2;
using Matrix_t = vector<vector<long long>>;
Matrix_t mul(Matrix_t &a, Matrix_t &b) {
    const int m = a.size(), n = b[0].size();
    Matrix_t mat = Matrix_t(m, vector<long long>(n, infll));
    for(int k = 0; k < n; ++k) {
        for(int i = 0; i < n; ++i) {
            for(int j = 0; j < n; ++j) {
                mat[i][j] = min(mat[i][j], a[i][k] + b[k][j]);
            }
        }
    }
    return mat;
}

Matrix_t mat_qpow(Matrix_t &x, int n) {
    Matrix_t ans = x;
    n--;
    while(n) {
        if(n & 1) ans = mul(x, ans);
        x = mul(x, x);
        n >>= 1;
    }
    return ans;
}

void solve() {
    int n, k;
    cin >> n >> k;
    Matrix_t cost(n, vector<ll>(n));
    for(int i = 0; i < n; ++i) {
        for(int j = 0; j < n; ++j) {
            cin >> cost[i][j];
        }
    }
    cost = mat_qpow(cost, k);
    for(int s = 0; s < n; ++s) cout<<cost[s][s]<<endl;
}
```
