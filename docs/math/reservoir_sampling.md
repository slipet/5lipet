# Reservoir Sampling 水塘抽樣演算法

在未知總體數量 $n$（或數據流為無限長）的情況下，從中隨機選取 $k$ 個樣本，使得每個元素被選中的機率均為 $\frac{k}{n}$。

方法:

1. 前 $k$ 個元素直接存入水塘。

2. 對於第 $i$ 個元素（$i > k$），以機率 $\frac{k}{i}$ 將其存入水塘。如果被選中，則從水塘中隨機替換掉一個既有元素。

```cpp
for(int i = k; i < n; ++i) {
    int r = random() % i;
    if(r < k) ans[r] = nums[i];
}
```
